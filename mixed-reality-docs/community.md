## <a name="microsoft-mixed-reality-mvps"></a>Microsoft Mixed Reality MVP

Microsoft 最有价值专家（Mvp）是 passionately 与社区分享他们的知识的技术专家。 它们始终处于 "不规则边缘"，并有一 unstoppable，可以让他们使用全新的、激动人心的技术。 他们对 Microsoft 产品和服务有很深入的了解，同时还能够将各种平台、产品和解决方案组合在一起，以解决实际问题。 Mvp 共同构成了跨90个国家/地区的4000多个技术专家和社区主管的全球社区，并由其热情、社区精神和寻求知识驱动。 除了其令人惊叹的技术能力外，Mvp 始终都愿意帮助他人-这就是它们的与众不同之处。
